
# Superstore-Sales-Analysis
## Project Overview
* Goal: To analyze sales data, identify trends, and provide actionable business insights.
* Tools Used: Power BI.
## Key Visuals:
~ Sales by Category.
~ Sales by Region.
~ KPI cards for total Sales and Profit.
~ Slicers for Segment-based filtering.
## Key Insights
~ The dashboard allows users to filter data dynamically by segment (Consumer, Corporate, Home Office).
~ It provides a clear overview of overall business performance in 2026.
